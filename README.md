# Mini-Sitio

Mini-sitio de práctica para aprender Git y GitHub.

## Qué contiene
- Página principal (index.html)
- Estilos en css/
- Imágenes en img/

## Cómo ejecutar localmente
1. Clonar el repo: `git clone https://github.com/tu-usuario/mini-sitio.git`
2. Entrar: `cd mini-sitio`
3. Abrir `index.html` en el navegador o ejecutar `npm run dev` si aplica
